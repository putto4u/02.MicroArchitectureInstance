## DBeaver 다운로드 및 설치 가이드

DBeaver는 Java 기반의 범용 데이터베이스 도구로, 설치가 간편하며 윈도우, 맥, 리눅스 등 다양한 환경을 지원합니다.

### 1. 프로그램 다운로드 및 설치

1. **공식 웹사이트 접속**: [DBeaver 공식 다운로드 페이지](https://dbeaver.io/download/)에 접속합니다.
2. **버전 선택**: 개인 학습 및 일반 업무용인 **'Community Edition'**을 선택합니다.
3. **OS별 파일 다운로드**:
* **Windows**: `Windows 64 bit (installer)` 파일을 권장합니다.
* **macOS**: 인텔 칩은 `macOS (x86_64)`, 애플 실리콘(M1/M2/M3)은 `macOS (Apple Silicon)`을 선택합니다.


4. **설치 진행**: 다운로드한 파일을 실행하여 설치 마법사의 안내에 따라 진행합니다. (기본 설정으로 `Next`를 눌러 완료하면 됩니다.)

## DBeaver 상세 사용 설명서

설치가 완료되었다면, 서버(온프레미스 또는 RDS)와 연결하여 실제 데이터를 관리하는 방법을 안내합니다.

### 1. 데이터베이스 연결 설정 (Connection)

1. **새 연결 시작**: 왼쪽 상단의 **'플러그 모양 아이콘'**을 클릭합니다.
2. **대상 DB 선택**: 목록에서 **MySQL**을 선택하고 `다음(Next)`을 누릅니다.
3. **상세 정보 입력**:
* **Server Host**: 서버 IP 주소 (본인 컴퓨터면 `127.0.0.1`)
* **Port**: `3306` (MySQL 기본값)
* **Database**: 접속할 DB 이름 (예: `employees`)
* **Username/Password**: 서버 설정 시 생성한 계정 정보 (예: `admin` / `password123`)


4. **드라이버 설치**: 처음 접속 시 드라이버가 없다는 알림이 뜨면 `Download`를 클릭하여 자동 설치합니다.
5. **연결 테스트**: `Test Connection` 버튼을 눌러 연결 성공을 확인한 후 `완료(Finish)`합니다.

### 2. 주요 인터페이스 구성

* **Database Navigator (좌측)**: 생성된 데이터베이스, 테이블, 뷰 등을 트리 구조로 탐색합니다.
* **SQL Editor (중앙)**: 직접 쿼리를 작성하는 창입니다. `Ctrl + Enter`로 실행합니다.
* **Data Grid (하단)**: 쿼리 결과나 테이블의 데이터를 엑셀 형태로 보여줍니다.

### 3. 데이터 조회 및 편집 활용법

1. **테이블 데이터 보기**: Navigator에서 테이블을 마우스 오른쪽 버튼으로 클릭한 뒤 `데이터 보기(View Data)`를 선택합니다.
2. **데이터 직접 수정**: 그리드 창에서 값을 직접 수정하고 하단의 **`Save` (또는 `Ctrl + S`)**를 누르면 DB에 즉시 반영됩니다.
3. **데이터 내보내기**: 결과창에서 우클릭 후 `Export Data`를 선택하면 **CSV, Excel, SQL Insert** 문 등으로 저장할 수 있습니다.

| 항목 | 상세 내용 | 유료 여부 |

# Chapter 6-2. DBeaver 초기 설정 및 DB 연결 실습

DBeaver를 처음 설치하고 실행했을 때, 데이터베이스 서버(MySQL)와 연결(Connection)을 맺는 구체적인 절차입니다. 가상 머신(Ubuntu)에 설치된 MySQL을 타겟으로 설정하는 과정을 다룹니다.

---

### 1. 새 연결 마법사 실행 (New Connection Wizard)

DBeaver 실행 후 가장 먼저 해야 할 일은 '어떤 데이터베이스에 접속할지' 정의하는 것입니다.

1. **메뉴 선택:** 상단 메인 메뉴바에서 **[Database]**를 클릭합니다.
2. **하위 메뉴:** 드롭다운 메뉴 중 **[New Database Connection]**을 클릭합니다.
* *(단축키 아이콘: 좌측 상단의 '플러그 모양 아이콘(New Connection)'을 클릭해도 동일합니다.)*



### 2. 데이터베이스 드라이버 선택

DBeaver는 수많은 DB를 지원하므로, MySQL을 사용하겠다고 명시해야 합니다.

1. **검색 또는 선택:** 팝업창(Connect to a database)이 뜨면, **'MySQL'** 아이콘을 찾아서 클릭합니다.
* *(목록이 많다면 상단 검색창에 'mysql'을 입력하여 필터링합니다.)*


2. **다음 단계:** 하단의 **[Next >]** 버튼을 클릭합니다.

### 3. 연결 상세 정보 입력 (Main 탭)

가장 중요한 단계입니다. 접속하려는 서버의 IP 주소와 계정 정보를 입력합니다.

| 입력 필드 | 입력값 (설명) |
| --- | --- |
| **Server Host** | **가상 머신 IP** 또는 **127.0.0.1** <br>

<br> (1) NAT 포트 포워딩 방식: `127.0.0.1` 또는 `localhost` <br>

<br> (2) 브리지 네트워크 방식: `192.168.x.x` (우분투 IP) |
| **Port** | **3306** (MySQL 기본 포트) |
| **Database** | (공란 가능) 접속 시 기본으로 사용할 DB명. 비워두면 전체 목록이 보입니다. |
| **Username** | 접속할 ID (예: `root` 또는 생성한 유저명) |
| **Password** | 설정한 비밀번호 |

### 4. (중요) 드라이버 설정 및 보안 옵션

최신 버전의 MySQL 접속 시 보안 설정 문제로 연결이 거부될 수 있습니다. 다음 설정을 미리 확인합니다.

1. **Driver Settings 탭 이동:** 입력창 상단의 탭 중 **[Driver Properties]** (또는 Edit Driver Settings)를 클릭합니다.
2. **allowPublicKeyRetrieval 설정:**
* 속성 목록에서 **`allowPublicKeyRetrieval`** 항목을 찾습니다.
* 값을 **`false`**에서 **`true`**로 변경합니다. (우측 Value 칸을 클릭하여 변경)
* *이 설정이 `false`일 경우 'Public Key Retrieval is not allowed'라는 에러가 발생할 수 있습니다.*



### 5. 드라이버 다운로드 및 연결 테스트

DBeaver는 처음 접속하는 DB의 경우, 필요한 자바 드라이버(.jar 파일)를 자동으로 다운로드합니다.

1. **Test Connection 클릭:** 설정창 좌측 하단의 **[Test Connection ...]** 버튼을 클릭합니다.
2. **드라이버 다운로드 팝업:** "Download driver files?" 팝업이 뜨면 **[Download]**를 클릭합니다. (최초 1회만 수행)
3. **성공 확인:** "Connected"라는 메시지와 함께 서버 버전 정보가 뜨면 연결 성공입니다.
4. **완료:** **[OK]** -> **[Finish]**를 눌러 설정을 저장합니다.

### 6. SQL 편집기(Script) 열기 및 작업 시작

연결이 완료되면 좌측 'Database Navigator' 탐색기에 코끼리 아이콘(또는 비버 아이콘)과 함께 연결 이름이 생성됩니다.

1. **연결 활성화:** 생성된 연결 아이콘을 더블 클릭하거나 화살표(>)를 눌러 트리를 펼칩니다. (초록색 체크 표시가 생기면 접속된 상태)
2. **SQL 편집기 실행:**
* 연결 이름 위에서 **[마우스 우클릭]** -> **[SQL Editor]** -> **[Open SQL Script]** 선택.
* 또는 상단 툴바의 **[SQL]** 아이콘 클릭.
* 단축키: **`F3`**


3. **쿼리 실행:**
* 빈 편집창에 `SELECT NOW();` 라고 입력해 봅니다.
* 실행 단축키: **`Ctrl` + `Enter**` (현재 커서가 있는 줄 실행)
